# Workout History Mobile QA

## Setup

- Generate a plan with training exercises.
- Open Plan Details.
- Start a workout.
- Complete some or all sets.
- Finish the workout.

## Training Entry

- Open Training tab.
- Tap Workout history.
- Confirm Workout History opens.
- Confirm completed workouts appear newest first.

## Profile Entry

- Open Profile.
- In Personal, find Completed workouts.
- Tap Workout history.
- Confirm the same history list opens.

## Today Summary

- Open Today after completing today's workout.
- Confirm a compact Workout completed card appears.
- Confirm it shows focus, set count, and Partial workout saved when partial.
- Tap View summary.
- Confirm the read-only workout detail opens.

## History List

- Confirm each row has date, focus, set count, exercise count, and partial badge when relevant.
- Confirm empty state appears for a user with no completed workouts.
- Confirm rows open completed details.

## Completed Details

- Confirm the screen says Workout completed.
- Confirm start/completed times and counts show.
- Confirm set states are visible but not editable.
- Confirm no Finish workout button is visible.
- Confirm Exercise Details opens for library-backed exercises.
- Confirm legacy free-text exercises render without Exercise Details.

## Safety And Copy

- Confirm partial completion wording is neutral.
- Confirm no shame/failure wording appears.
- Confirm no raw debug, protocol, user ID, or DailyPlan JSON appears.
